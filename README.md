# 🎓 MIT University Homepage Clone

## 📌 Overview
This project is a front-end clone of the official Massachusetts Institute of Technology (MIT) homepage. It was built to practice modern web development skills, including layout design, responsiveness, and UI replication.

The goal was to closely match the structure, styling, and user experience of the original website.

---

## 🚀 Features
- Responsive homepage layout
- Clean and structured navigation bar
- Hero section with featured content
- Sections for news, events, and highlights
- Footer with organized links and information
- UI closely resembling the original MIT website

---

## 🛠️ Tech Stack
- HTML5  
- CSS3
